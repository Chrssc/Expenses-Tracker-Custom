# Expenses-Tracker-Custom
Building Customise Expenses Tracker that can consistently tracking expenses throughout months, or years.

## About

One day I checked my bank and realised how my money could have gone so fast. I tried using notes, Excel (spreadsheets) and also other apps that are available, but I still couldn't consistently track my budget. Up to one day (1 February 2026), I decided to create a Google Form that only required very little info (amount, category, payment method and brief description — optional). My goal for the Google Form was to track my expenses immediately, taking an average of 10 seconds or less.

Marked 1 March 2026, I realised that I managed to consistently keep track of my expenses — which brought me to this project.

The purpose of this project is:

- Know how much I spent
- Automated analysis
- Transactions history
- Options to add my own categories and payment methods
- View my transactions
- Most importantly, keep track of my expenses so that I can plan my finances more efficiently

This is planned as a **web app specially designed for mobile**.
